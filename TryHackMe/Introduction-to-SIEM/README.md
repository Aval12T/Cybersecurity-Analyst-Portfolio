# Introduction to SIEM

## Overview

This TryHackMe room introduced Security Information and Event Management (SIEM) concepts, including log sources, log ingestion, event monitoring, alerting, and detection rules.

## Skills Demonstrated

- SIEM fundamentals
- Log source identification
- Log ingestion methods
- Windows Event Log analysis
- Security event monitoring
- Detection rule creation
- Alert investigation
