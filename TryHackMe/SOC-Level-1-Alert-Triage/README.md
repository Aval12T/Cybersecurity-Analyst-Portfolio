# SOC Level 1 Alert Triage

## Overview

This TryHackMe room introduced the workflow of a Security Operations Center (SOC) analyst and the process of reviewing, investigating, and escalating security alerts.

## Skills Demonstrated

- Alert triage
- Security monitoring
- Log analysis
- Incident investigation
- Escalation procedures

## Key Concepts

### Alert Classification
Reviewed security alerts and identified their severity and priority.

### Investigation Process
Analyzed alert details and supporting evidence to determine legitimacy.

### Escalation
Learned when and how incidents should be escalated to higher-level analysts.

## Tools and Technologies

- SIEM Concepts
- Security Event Logs
- Alert Management Workflows

## Lessons Learned

This room improved my understanding of SOC operations and strengthened my ability to evaluate security alerts and investigate potential incidents.

## Career Relevance

The skills practiced in this lab directly relate to Security Operations Center (SOC) Analyst and Cybersecurity Analyst responsibilities.
