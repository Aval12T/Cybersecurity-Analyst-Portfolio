# Elastic Basics

## Overview

This TryHackMe room introduced Elastic Security and demonstrated how analysts use Elasticsearch and Kibana to collect, search, visualize, and investigate security data. The room covered Elastic architecture, data ingestion, and security monitoring concepts.

## Skills Demonstrated

* Elastic fundamentals
* Kibana navigation
* Elasticsearch concepts
* Log analysis
* Security monitoring
* Data visualization
* Threat hunting concepts

## Key Takeaways

* Learned the core components of the Elastic Stack.
* Explored how Kibana is used to visualize and investigate security data.
* Reviewed methods for ingesting and analyzing log data.
* Developed familiarity with a SIEM platform commonly used by SOC Analysts.
* Gained experience navigating dashboards and security monitoring workflows.
